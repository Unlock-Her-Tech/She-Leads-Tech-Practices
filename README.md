# She-Leads-Tech-Practices(Typescript& Javascript)

## Overview

**She Leads Tech Practices** is the technical component of [Unlock Her Tech](https://unlockhertech.com/), dedicated to building problem-solving skills through hands-on coding sessions.

We host **live coding sessions every 2 weeks** where we tackle real coding challenges, discuss algorithms, and strengthen our fundamentals together.

### Join Us

For details about upcoming sessions, times, and to RSVP, visit: [She Leads Tech Practice on Luma](https://luma.com/sheleadstechpractice)

---

## Our Approach

**Structure is the Solution to the Problem.**

We believe that breaking down complex problems into manageable parts is key to becoming a better engineer. Each session focuses on:

- Understanding the problem
- Identifying patterns and constraints
- Building structured, scalable solutions
- Learning from each other's approaches
