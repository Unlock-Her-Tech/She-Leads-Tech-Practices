### 2️⃣ JavaScript `Map`

A `Map` stores information as **key-value pairs**. It is especially useful for counting values, tracking frequencies and solving problems such as Valid Anagram.

**JavaScript Map introduction:**
https://www.w3schools.com/js/js_maps.asp

**JavaScript Map methods:**
https://www.w3schools.com/js/js_map_methods.asp

**Official MDN Map documentation:**
https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Map

**MDN guide to Maps and keyed collections:**
https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Keyed_collections

Focus on these methods:

- `set()`
- `get()`
- `has()`
- `delete()`
- `keys()`
- `values()`
- `entries()`

### 3️⃣ Session 1 LeetCode Problem: Valid Anagram

🧩 **Problem:**
https://leetcode.com/problems/valid-anagram/

Try solving it independently before looking at the explanation.

Think about:

- How can we count each character?
- How can a `Map` help us?
- What should happen when the strings have different lengths?
- What are the time and space complexities?

**Explanation and different approaches:**
https://neetcode.io/solutions/valid-anagram
